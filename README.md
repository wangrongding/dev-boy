# dev-boy

一个超级强大的 cli 工具集，快速在终端完成一些常用的操作。

## 包含的功能

- [x] 获取本机外网 ip 地址
- [x] 获取本机内网 ip 地址
- [x] 获取本机 mac 地址
- [x] 获取本机系统相关信息
- [x] 打印当前目录下的文件树
- [ ] 终端快速打开地图
- [ ] 终端快速打开文档
- [ ] 终端快速搜索
- [ ] 生成 gitignore 文件模版
